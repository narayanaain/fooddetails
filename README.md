# Fitness_partner

BACKEND

Nodejs + Express:
clone the file
open in terminal
run 'npm i'
follow up with 'npm start'
note: .env file must be set for working MongoDB database

FRONTEND

React + Vite:
clone the file
open in terminal
run 'npm i'
follow up with 'npm run dev'
note: .env file must be set for working Backend url
